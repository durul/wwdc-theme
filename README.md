#WWDC 2016 Theme

A color scheme for Xcode based on the WWDC 2016 invitation. Enjoy!

###Installation
This extension requires Brackets Release 1.8 or newer.

- Open Brackets
- Open the Extension Manager
- Switch to "Themes" tab
- Search for "WWDC Theme"
- Click "Install"
